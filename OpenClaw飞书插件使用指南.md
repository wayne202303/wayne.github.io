你的 OpenClaw 已经配置了 **5 个飞书插件**：

| 插件 | 功能 | 状态 |

## 💬 聊天插件 (feishu_chat)

### 功能说明
- 发送和接收飞书消息
- 查看聊天记录
- 搜索消息
- 管理群聊

### 使用示例

#### 1. 查看聊天列表
```text
# 在 OpenClaw 对话中说：
"列出我最近的聊天"
```

#### 2. 查看指定聊天记录
```text
"查看 MK团建（闲聊）信息群 最近 50 条消息"
```

#### 3. 发送消息
```text
"给王楠发送消息：你好"
```

⚠️ **注意**：发送消息可能需要 `im:message.send_as_user` 权限。

#### 4. 搜索消息
```text
"搜索包含 记忆系统 的消息"
```

## 🔧 与 lark-cli 的配合使用

OpenClaw 的飞书插件和 `lark-cli` 可以配合使用：

### 什么时候用哪个？

| 场景 | 推荐工具 |
|------|---------|
| **自然语言交互** | OpenClaw 飞书插件 |
| **批量操作/脚本** | lark-cli |
| **查看详细状态** | lark-cli |
| **简单对话式操作** | OpenClaw 飞书插件 |

### 示例：混合使用

```bash
# 先用 lark-cli 查看状态
lark-cli auth status

# 然后在 OpenClaw 中用自然语言操作
"给我看看下周的会议"
```

---

## 💡 实际使用建议

1. **从简单开始** - 先试试查看日历、搜索文档等基础功能
2. **查看状态** - 经常用 `openclaw status` 检查飞书插件状态
3. **查看日志** - 遇到问题时用 `openclaw logs --follow` 查看
4. **权限问题** - 如果某个功能用不了，可能是权限不够
