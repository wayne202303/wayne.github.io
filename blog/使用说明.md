## 快速开始

### 3步开始使用

1. **打开博客**
   - 双击 `index.html` 文件
   - 或使用本地服务器打开

2. **浏览文章**
   - 查看首页文章列表
   - 点击文章标题查看详情
   - 使用分页浏览更多文章

3. **搜索和筛选**
   - 使用搜索框查找文章
   - 点击分类筛选文章
   - 点击标签筛选文章

## 使用指南

### 浏览文章

#### 在首页浏览

1. **查看文章列表**
   - 打开博客首页
   - 滚动浏览文章卡片
   - 每篇文章显示：
     - 📅 发布时间
     - 📁 分类
     - 文章标题
     - 文章摘要
     - 标签列表

2. **使用分页**
   - 点击页码跳转到对应页
   - 点击"上一页"/"下一页"导航
   - 当前页高亮显示

#### 查看文章详情

1. **打开文章**
   - 点击文章标题
   - 或点击文章卡片任意位置
   - 文章内容会完整显示

2. **文章详情页包含**
   - 文章大标题
   - 发布时间和分类
   - 标签列表
   - 完整的文章内容（Markdown 渲染）
   - 返回列表按钮

3. **返回列表**
   - 点击"← 返回列表"按钮
   - 或使用浏览器的返回按钮

### 搜索文章

#### 使用搜索框

1. **输入搜索词**
   - 在顶部搜索框输入关键词
   - 支持中文和英文搜索
   - 搜索不区分大小写

2. **执行搜索**
   - 点击"搜索"按钮
   - 或按回车键

3. **查看结果**
   - 搜索结果会显示在文章列表区域
   - 包含标题、内容或摘要匹配的文章
   - 清空搜索框显示所有文章

#### 搜索技巧

- **搜索标题**：输入文章标题中的关键词
- **搜索内容**：输入文章正文中的关键词
- **组合搜索**：多个关键词用空格分隔（简单的AND搜索）

### 按分类筛选

#### 使用分类筛选

1. **查看分类列表**
   - 在侧边栏找到"分类"部分
   - 显示所有文章分类
   - 每个分类显示文章数量

2. **点击分类筛选**
   - 点击任意分类
   - 只显示该分类下的文章
   - 分页仍然有效

3. **清除筛选**
   - 点击"首页"导航链接
   - 或使用搜索框清空搜索
   - 显示所有文章

### 按标签筛选

#### 使用标签筛选

1. **查看标签列表**
   - 在侧边栏找到"标签"部分
   - 显示所有文章标签
   - 标签按使用频率排序

2. **点击标签筛选**
   - 点击任意标签
   - 只显示包含该标签的文章
   - 可以连续点击多个标签（每次只筛选一个）

3. **查看最新文章**
   - 在侧边栏找到"最新文章"部分
   - 显示最近的5篇文章
   - 快速访问最新内容

## 数据编辑

### 添加新文章

#### 准备文章数据

1. **打开数据文件**
   - 找到 `data/posts.json` 文件
   - 使用文本编辑器打开（推荐 VS Code）

2. **添加文章对象**
   - 在数组末尾添加新的文章对象
   - 确保 JSON 格式正确
   - 使用逗号分隔文章对象

#### 文章字段说明

```json
{
    "id": 13,                          // 必需：唯一数字ID
    "title": "文章标题",               // 必需：文章标题
    "slug": "article-slug",            // 必需：URL友好标识
    "date": "2024-04-16T10:00:00",  // 必需：发布时间
    "category": "分类名称",            // 必需：文章分类
    "tags": ["标签1", "标签2"],       // 必需：标签数组
    "excerpt": "文章摘要",             // 必需：简短摘要
    "content": "# 标题\n\n内容"       // 必需：Markdown内容
}
```

#### 字段详细说明

**id (number)**
- 必须是唯一的数字
- 建议按顺序递增
- 示例：1, 2, 3, ...

**title (string)**
- 文章的完整标题
- 建议 5-30 个字
- 会显示在文章列表和详情页

**slug (string)**
- URL 友好的文章标识
- 只使用小写字母、数字、连字符
- 示例：`my-awesome-article`

**date (string)**
- ISO 8601 格式的日期时间
- 格式：`YYYY-MM-DDTHH:MM:SS`
- 示例：`2024-04-16T10:00:00`

**category (string)**
- 文章所属分类
- 建议使用已有分类，或创建新分类
- 示例：`前端开发`, `后端开发`, `数据分析`

**tags (array)**
- 相关标签的数组
- 可以有多个标签
- 示例：`["Vue", "JavaScript", "前端框架"]`

**excerpt (string)**
- 文章的简短摘要
- 建议 50-200 个字
- 显示在文章列表中

**content (string)**
- 完整的文章内容
- 支持 Markdown 格式
- 可以包含代码块、列表、引用等

### 编辑现有文章

1. **找到文章**
   - 在 `data/posts.json` 中找到对应文章
   - 通过 id 或 title 定位

2. **修改字段**
   - 修改需要更新的字段
   - 保持 JSON 格式正确
   - 保存文件

3. **验证更改**
   - 刷新浏览器查看效果
   - 确认更改已生效

### 删除文章

1. **找到文章**
   - 在 `data/posts.json` 中定位文章

2. **删除对象**
   - 删除整个文章对象
   - 删除多余的逗号
   - 确保 JSON 格式正确

3. **验证删除**
   - 刷新浏览器
   - 确认文章已不在列表中

### Markdown 语法指南

#### 标题

```markdown
# 一级标题
## 二级标题
### 三级标题
```

#### 文本格式

```markdown
**粗体文本**
*斜体文本*
`行内代码`
```

#### 列表

```markdown
- 无序列表项1
- 无序列表项2

1. 有序列表项1
2. 有序列表项2
```

#### 代码块

\`\`\`javascript
function hello() {
    console.log('Hello!');
}
\`\`\`

#### 引用

```markdown
> 这是引用文本
> 可以有多行
```

#### 链接

```markdown
[链接文字](https://example.com)
```

## 自定义配置

### 修改博客标题

1. 打开 `index.html`
2. 找到 `<h1 class="blog-title">`
3. 修改其中的文字
4. 保存文件

### 修改配色方案

1. 打开 `css/style.css`
2. 找到 `:root` 部分
3. 修改 CSS 变量值
4. 保存文件

示例：

```css
:root {
    --primary-color: #e74c3c;      /* 主色调改为红色 */
    --secondary-color: #34495e;    /* 次要颜色 */
    /* ... 其他变量 */
}
```

### 修改导航栏

1. 打开 `index.html`
2. 找到 `<nav class="nav">` 部分
3. 添加或修改导航链接
4. 保存文件

### 修改每页文章数量

1. 打开 `js/app.js`
2. 找到 `this.postsPerPage = 10;`
3. 修改数字为想要的数量
4. 保存文件

## 常见问题

### 一般问题

**Q: 博客打不开怎么办？**

A: 请检查：
1. 文件是否完整
2. 是否使用了本地服务器（推荐）
3. 浏览器控制台是否有错误（按 F12）

**Q: 文章不显示？**

A: 请检查：
1. `data/posts.json` 文件是否存在
2. JSON 格式是否正确
3. 浏览器控制台是否有错误

**Q: 搜索功能不工作？**

A: 请检查：
1. JavaScript 文件是否正确加载
2. 搜索框的事件绑定是否正常
3. 浏览器控制台是否有错误

### 数据问题

**Q: 如何验证 JSON 格式？**

A: 使用在线工具：
- https://jsonlint.com/
- https://jsonformatter.curiousconcept.com/

**Q: 添加文章后不显示？**

A: 请检查：
1. id 是否唯一
2. JSON 格式是否正确
3. 是否保存了文件
4. 是否刷新了浏览器

**Q: Markdown 不渲染？**

A: 请检查：
1. Marked 库是否正确加载
2. 文章内容格式是否正确
3. 浏览器控制台是否有错误

### 样式问题

**Q: 样式显示不正确？**

A: 请检查：
1. CSS 文件路径是否正确
2. 浏览器缓存是否清除（Ctrl+F5）
3. 是否有 CSS 语法错误

**Q: 移动端显示不正常？**

A: 请检查：
1. viewport meta 标签是否存在
2. 媒体查询是否正确
3. 使用浏览器的开发者工具模拟移动端

### 性能问题

**Q: 博客加载很慢？**

A: 优化建议：
1. 使用本地服务器而不是直接打开文件
2. 压缩 CSS 和 JavaScript 文件
3. 优化图片大小
4. 使用 CDN 加载外部库
