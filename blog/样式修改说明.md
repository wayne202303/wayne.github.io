## 🎨 设计概述

本次样式重构参考了虾评Skill网站的设计风格，打造了一个**简约大气**的博客界面。

### 设计理念
- **简约** - 减少视觉干扰，突出内容
- **大气** - 充足留白，舒适阅读体验
- **现代** - 卡片式布局，清晰层次
- **专业** - 统一间距系统，规范字体

### 设计原则
1. **内容优先** - 内容是主角，装饰是配角
2. **视觉层次** - 通过大小、颜色、间距建立清晰层次
3. **一致性** - 统一的设计语言和交互模式
4. **响应式** - 适配各种屏幕尺寸

## ✨ 设计特点

### 1. 配色系统
- **主色调** - 经典黑白灰
- **点缀色** - 红色作为强调色
- **背景色** - 纯白背景，浅灰辅助
- **边框色** - 淡灰色边框

### 2. 字体系统
- **主字体** - 系统无衬线字体
- **字重** - 从细到粗7级
- **字间距** - 优化阅读体验

### 3. 间距系统
- **8px基准** - 所有间距都是8的倍数
- **卡片间距** - 24px卡片间距
- **内容间距** - 16px内容间距

### 4. 圆角系统
- **小圆角** - 6px按钮圆角
- **中圆角** - 12px卡片圆角
- **大圆角** - 16px容器圆角

### 5. 阴影系统
- **小阴影** - 轻微悬浮
- **中阴影** - 卡片阴影
- **无外发光** - 现代简洁风格

## 🎯 颜色系统

### 主要颜色
```css
:root {
    --primary-color: #1a1a1a;      /* 主色：深黑 */
    --secondary-color: #666666;    /* 次要色：中灰 */
    --accent-color: #eb3349;       /* 强调色：红色 */
    --accent-hover: #d62d41;      /* 强调色悬停 */
    --bg-color: #ffffff;           /* 背景：纯白 */
    --bg-secondary: #f8f8f8;       /* 次背景：浅灰 */
    --border-color: #e5e5e5;        /* 边框：淡灰 */
    --text-primary: #1a1a1a;        /* 主文字：深黑 */
    --text-secondary: #666666;      /* 次文字：中灰 */
    --text-muted: #999999;           /* 弱化文字：浅灰 */
}
```

### 颜色使用规则
- **主文字** - 标题、重要信息
- **次文字** - 正文、描述
- **弱化文字** - 时间、辅助信息
- **强调色** - 链接、悬停、重要操作

## 📐 布局结构

### 整体布局
```
┌─────────────────────────────────────┐
│         Header (固定顶部)            │
├──────────┬──────────────────────────┤
│ 侧边栏  │   主内容区 (3列网格)  │
│        │                          │
│        │                          │
│        │                          │
│        │                          │
└────────┴──────────────────────────┘
│         Footer                    │
└─────────────────────────────────────┘
```

### 响应式断点
| 屏幕尺寸 | 布局 |
|----------|------|
| ≥ 1024px | 侧边栏 + 主内容区 |
| < 1024px | 单列布局，侧边栏在上 |
| < 768px | 单列布局，优化间距 |

### 卡片网格
- **桌面端** - 3列卡片网格
- **平板端** - 2列卡片网格
- **手机端** - 1列卡片网格

## 📱 响应式设计

### 断点设置
```css
/* 桌面端 */
@media (max-width: 1024px) {
    .content-wrapper {
        grid-template-columns: 1fr;
    }
}

/* 平板端 */
@media (max-width: 768px) {
    .container {
        padding: 0 16px;
    }
    .articles-grid {
        grid-template-columns: 1fr;
    }
}
```

### 移动端优化
- **触控友好** - 足够的点击区域
- **字体缩放** - 合适的阅读大小
- **间距调整** - 舒适的触摸间距
- **性能优化** - 快速的加载速度

## 🚀 使用指南

### 快速开始
1. **访问简约版首页
```
http://localhost:8000/index-minimal.html
```

2. **切换到简约版样式
修改 `index-minimal.html` 中的 CSS 引用：
```html
<!-- 旧版本 -->
<link rel="stylesheet" href="css/style.css">

<!-- 新版本 -->
<link rel="stylesheet" href="css/style-minimal.css">
```

### 自定义配置

#### 修改主题色
在 `css/style-minimal.css` 中修改 CSS 变量：
```css
:root {
    --accent-color: #你的颜色;
}
```

#### 修改间距
```css
:root {
    --radius-md: 8px;  /* 调整圆角 */
}
```
