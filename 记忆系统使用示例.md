> 演示：下载并安装另一个技能的完整流程

---

## 📋 任务目标

从虾评Skill平台下载并安装另一个热门技能，完整演示记忆系统的使用流程。

## 🔄 完整流程演示

### 第一步：任务开始 - 读取记忆

**每次对话开始时，先读取以下文件：**

1. **读取 SESSION-STATE.md** - 了解当前任务状态
2. **读取最近 1-3 天 daily notes** - 了解最近发生的事情
3. **最后才做 memory_search** - 按需检索长期记忆

**示例读取结果：**

### 第三步：任务结束前 30 秒 - 使用 memory-capture

**先整理候选记忆，不直接写 MEMORY.md：**

```markdown

- 选择技能时优先看下载量和评分，这个策略有效

- 每次下载后更新 memory.md 中的已安装技能列表

- 下载技能时要注意剩余虾米数量
```

## 📊 完整流程图

```
任务开始
    ↓

    ↓
读取最近 daily notes
    ↓
（可选）memory_search
    ↓
┌─────────────────┐
│  任务进行中      │
│  ↓              │
│  working-buffer │
│  （毛坯记录）    │
└─────────────────┘
    ↓

    ↓
┌──────────────────┐
│  memory-capture  │
│  （候选记忆）     │
└──────────────────┘
    ↓
蒸馏到 MEMORY.md
    ↓
更新 daily note
    ↓
更新 SESSION-STATE.md
    ↓
任务结束
```

---

## 💡 关键原则

1. **先写毛坯，再蒸馏** - 不要一开始就追求完美
2. **分层记录** - 不同类型的记忆放在不同文件
3. **保持精简** - MEMORY.md 只放真正长期有用的
4. **定期回顾** - 每周蒸馏一次 daily notes
