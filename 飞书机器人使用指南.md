## 飞书开放平台配置

### 步骤1：登录飞书开放平台
访问：https://open.feishu.cn/

### 步骤2：进入应用管理
1. 点击右上角"控制台"
2. 选择"应用管理"
3. 找到您的应用（App ID: `cli_a95276171379dbda`）

### 步骤3：配置应用权限
在应用详情页中：
1. 点击"权限管理"
2. 添加以下权限：
   - `im:message` - 发送消息
   - `im:message.group` - 群聊消息
   - `im:message.p2p` - 单聊消息
   - `bot` - 机器人相关权限

### 步骤4：发布应用
1. 点击"版本管理与发布"
2. 创建新版本并发布
3. 等待审核通过（或设置为企业内部应用）

## 使用飞书 CLI

### 安装飞书 CLI
如果还没有安装飞书 CLI，请运行：
```bash
lark-cli config init
```

### 登录飞书 CLI
```bash
lark-cli auth login
```

### 验证配置
```bash
lark-cli api GET /open-apis/authen/v1/user_info
```

## 运行机器人脚本

### 方式1：使用简单版（推荐）

这是最简单的方式，提供交互式菜单：

```bash
python feishu_bot_simple.py
```

选项说明：
- `1` - 执行 Jira 同步（仅本地运行）
- `2` - 执行 Jira 同步并将结果发送到飞书
- `3` - 退出程序

### 方式2：使用飞书 CLI 版本

这个版本使用飞书 CLI 来发送消息：

```bash
python feishu_cli_bot.py
```

选项说明：
- `1` - 执行 Jira 同步（仅本地运行）
- `2` - 执行 Jira 同步并将结果发送到飞书
- `3` - 测试发送飞书消息
- `4` - 退出程序

### 方式3：直接运行 Jira 同步

如果只想运行 Jira 同步，不使用机器人：

```bash
python .agents\skills\jira_to_feishu\scripts\jira_to_feishu.py
```

## 📞 常见问题

### Q: 如何获取我的 Open ID？
A: 可以通过飞书开放平台的 API 获取，或者让机器人给您发消息，从消息中获取。

### Q: 消息发送失败怎么办？
A: 请检查：
1. 应用是否已发布
2. 是否配置了正确的权限
3. Open ID 是否正确

### Q: 可以在群聊中使用吗？
A: 可以，需要配置群聊权限和获取群聊 ID。
